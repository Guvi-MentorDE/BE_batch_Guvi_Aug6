The process of inheriting the properties of the parent class into a child class is called inheritance. 
The existing class is called a base class or parent class and the new class is called a subclass or child class or derived class.

syntax:
-------

class BaseClass:
  Body of base class
class DerivedClass(BaseClass):
  Body of derived class

Types:
-------
Single inheritance    ; parent -> child 
Multilevel inheritance  parent -> child 1 -> child 2 

Multiple Inheritance  ; parent1 , parent2 -> child 
Hierarchical Inheritance ; parent -> child 1, child2 , child3
Hybrid Inheritance ; parent -> child1 , child2 -> child 3
